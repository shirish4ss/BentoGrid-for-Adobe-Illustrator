## Features
- 50+ preset bento grid patterns across 1–6 columns
- Live ASCII preview of grid layout before generating
- 20 curated color palette presets (Dark, Light, Vibrant, Pastel, Monochrome & more)
- Random pattern generator (Shuffle button)
- Custom pattern input — Simple mode and advanced JSON mode
- Color fill with Hex input and Illustrator foreground auto-detect
- Stroke styling — enable, width, and RGB color
- Rounded corners toggle with custom radius
- Background frame with padding and custom color
- Multi-color cell cycling across palette shades
- Save / Load settings to JSON file
- Auto-syncs grid size from active Illustrator artboard
- Compatible with Adobe Illustrator CC 2021 and newer
***

## Getting Started

### 1. Installation
- Download `BentoGrid_ShirishSalve_v3.jsx` from the release.
- Place the script in your Illustrator Scripts folder:
  - **Windows:** `C:\Program Files\Adobe\Adobe Illustrator <version>\Presets\en_US\Scripts\`
  - **macOS:** `/Applications/Adobe Illustrator <version>/Presets/en_US/Scripts/`
- Restart Illustrator. The script will appear under `File > Scripts`.

### 2. Usage
- Open a document in Adobe Illustrator.
- Run the script from `File > Scripts > BentoGrid_ShirishSalve_v3`.
- The BentoGrid dialog will open with 6 tabs:

| Tab | What it does |
|---|---|
| **Grid & Pattern** | Set grid size, gutter, choose or randomize a pattern |
| **Live Preview** | ASCII map showing your current layout |
| **Color & Fill** | Choose palette preset or enter custom Hex / RGB |
| **Stroke & BG** | Rounded corners, stroke, background frame |
| **Options** | Naming, multi-color cycling, save/load settings |
| **About** | Bio, Behance link, support / donation |

- Configure your settings across the tabs.
- Click **Generate Bento Grid ▶** to place the grid on your artboard.

## Tabs Reference

### Grid & Pattern
- **Grid Size** — Width and height in px, auto-synced from the active artboard.
- **Gutter** — Space between cells. Quick presets: None (0px) to Spacious (64px).
- **Preset Pattern** — 50+ named layouts (e.g. Hero Left 3×2, Dashboard 4×3, Mosaic 3×3).
- **Random Pattern** — Generates a randomized spanning grid.
- **Custom Pattern** — Enter your own layout in Simple mode (`2,1 | 1,1,1`) or advanced JSON.

### Live Preview
- Displays an ASCII box-art map of the selected pattern.
- Shows cols × rows, gutter size, and artboard dimensions.
- Updates automatically whenever any setting changes.

### Color & Fill
- **Palette Presets** — 20 curated palettes: Deep Dark, Neon Pulse, Ocean Blue, Rose Blush, etc.
- **Auto-detect** — Picks up the current Illustrator foreground fill color automatically.
- **Manual RGB** — Enter values between 0 and 1.
- **Hex Input** — Enter any hex color (e.g. `#1A2B3C`) and click Apply Hex.

### Stroke & BG
- **Rounded Corners** — Enable and set corner radius in px.
- **Stroke** — Add a stroke to all cells; set width and RGB color.
- **Background Frame** — Adds a background rectangle behind the grid with custom padding and color.

### Options
- **Naming** — Set the group name and cell prefix (cells named `Prefix_Row_Col`, e.g. `Cell_0_0`).
- **Multi-Color** — Cycles palette shades across cells instead of using a single fill color.
- **Save / Load** — Saves all current settings to a JSON file in your system temp folder and reloads them next run.

### About
- Designer bio, Behance portfolio link, and Buy Me a Coffee support link.
- URLs are copyable directly from editable text fields in the dialog.
